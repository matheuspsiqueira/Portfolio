Sobre o projeto

Desenvolvi o software de automação na linguagem Python, utilizando algumas bibliotecas especializadas, como Tesseract e Fitz, para interpretação e tratamento de arquivos. Outras bibliotecas foram empregadas para transformá-lo em um executável, capaz de instalar automaticamente todos os pacotes necessários para funcionar em qualquer computador com sistema Windows. Um exemplo dessas bibliotecas é o PyInstaller. Além disso, todos os pacotes e dependências utilizados foram listados no arquivo requirements.txt para facilitar a instalação.

O software funciona assim:

➡️ "Divide" um arquivo PDF compilado em vários documentos;

➡️ Separa e organiza esses documentos em pastas nomeadas de acordo com cada colaborador;

➡️ Renomeia os arquivos de forma automatizada.
