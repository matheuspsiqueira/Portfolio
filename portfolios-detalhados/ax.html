Sobre o projeto

Esse projeto foi um dos primeiros no uso do Bootstrap. Para trazê-lo à vida, combinei elementos de HTML5, CSS3, JavaScript e, é claro, o Bootstrap.

A concepção por trás do projeto era criar um site dedicado à venda de relógios, mas com o intuito principal de explorar e aprimorar o front-end, deixando o back-end de lado por enquanto.
