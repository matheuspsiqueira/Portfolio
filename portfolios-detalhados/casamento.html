Sobre o projeto

Este projeto é um projeto pessoal 100% funcional, desenvolvido especialmente para celebrar um dos momentos mais importantes da minha vida: o meu casamento. Além de demonstrar minhas habilidades técnicas em desenvolvimento web, o site está em uso real neste momento, servindo como plataforma oficial para convidados, gestão de presentes e informações do evento.


Tecnologias Utilizadas:

  • Django (Python) como framework principal para o back-end;
  • HTML5, CSS3, JavaScript no front-end, com componentes dinâmicos e responsivos;
  • Banco de dados SQLite3 em ambiente de desenvolvimento e PostgreSQL para produção;
  • Integração com a API do Mercado Pago para sistema de pagamento totalmente funcional;
  • Deploy realizado em servidor Linux com Nginx como servidor web e Gunicorn como WSGI HTTP server.

Funcionalidades Principais:

    Lista de Presentes Interativa: Os noivos podem cadastrar os presentes manualmente no painel administrativo, com controle de estoque automático. Cada presente pode ser comprado uma única vez, com pagamento gerado em tempo real pela API do Mercado Pago.

    Sistema de Pagamento Seguro: Após a escolha do presente, o convidado é direcionado para uma página intermediária e, após o pagamento, o sistema identifica e atualiza o status automaticamente.

    Confirmação de Presença: Cada convidado pode confirmar sua presença de forma simples e rápida. As respostas são armazenadas no banco de dados e podem ser acessadas pelos noivos em um painel de controle privado.

    Mural de Mensagens: Disponibilizamos uma área para que os convidados possam deixar mensagens carinhosas para os noivos. As mensagens são salvas e enviadas aos noivos.

    Contador Regressivo: Um contador de dias destaca a chegada do "grande dia", reforçando a expectativa positiva para todos que acessam o site.

    Mapa com Geolocalização: A seção "Local do Evento" exibe um mapa interativo com GPS para facilitar o deslocamento até o local da cerimônia.

    Design Responsivo e Otimizado: O site foi pensado para oferecer uma experiência suave tanto em dispositivos móveis quanto em desktops, com foco na acessibilidade e usabilidade para pessoas de todas as idades.
