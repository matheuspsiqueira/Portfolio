Sobre o projeto

Trabalhando em equipe para o nosso projeto final do curso técnico, tive a oportunidade de assumir a responsabilidade pelo desenvolvimento do back-end. Estava imerso nos estudos do Django, o que me levou a escolher este framework para a tarefa.

Utilizamos HTML5, CSS3, JavaScript, Bootstrap e, é claro, o framework Django, juntamente com o banco de dados SQLite3 em estágios iniciais e posteriormente migrando para o PostgreSQL.

A primeira página visa mostrar mais informações sobre a loja e quais são seus diferenciais, a interação do usuário começa ao clicar em "Cadastro" na barra de navegação para a criação de uma conta, proporcionando um processo de cadastro simples e intuitivo.

O catálogo de roupas masculinas abrange uma ampla gama de estilos, proporcionando uma experiência de navegação agradável. A interface amigável facilita a exploração do catálogo, enquanto os filtros de categoria permitem aos usuários refinar suas escolhas, concentrando-se, por exemplo, em "Camisas".

A implementação de permissões de acesso destaca-se como um elemento do projeto. Apenas usuários autenticados têm acesso à página do carrinho de compras, garantindo a segurança e a integridade das transações. Essa abordagem não apenas protege a privacidade do usuário, mas também contribui para uma experiência de compra mais personalizada.

No âmbito do controle de estoque, introduzimos o MILL S.A., uma solução desenvolvida anteriormente para gerenciar com precisão o inventário da loja KingsMen's.

Por fim, vale ressaltar que todos os produtos estão salvos no banco de dados da loja, assim como as informações cadastradas dos clientes.
