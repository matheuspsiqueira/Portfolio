Sobre o projeto

Criei este projeto de forma ágil para servir como um exemplo de uma loja de roupas que poderia tirar proveito dos serviços da empresa fictícia MILL S.A. No desenvolvimento, empreguei HTML5, CSS3, Bootstrap e JavaScript.

A abordagem para o site é direta, concentrando-se principalmente no front-end de uma loja que atua como revendedora de roupas autorizadas. Com o uso de tecnologias como Bootstrap e JavaScript, busquei criar uma experiência visualmente agradável e interativa para os visitantes.

Embora tenha sido um projeto rápido, a intenção foi demonstrar a viabilidade de uma loja de roupas adotando os serviços da MILL S.A por meio de um design elegante e funcional.
