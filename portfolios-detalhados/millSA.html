Sobre o projeto

Trabalhando em equipe como parte de um projeto para o nosso curso técnico, tive a oportunidade de assumir a responsabilidade pelo desenvolvimento do back-end. Na época, estava imerso nos estudos do Django, o que me levou a escolher este framework para a tarefa.

Utilizamos HTML5, CSS3, JavaScript, Bootstrap e, é claro, o framework Django, juntamente com o banco de dados SQLite3 em estágios iniciais e posteriormente migrando para o PostgreSQL.

A essência do projeto residia na criação de um site ou sistema abrangente para o controle de estoque. A estratégia envolveu a concepção de uma empresa fictícia, a "MILL S.A", que atuava como uma fornecedora terceirizada de soluções de gerenciamento. Imaginamos que empresas de menor porte, como uma loja de roupas, optariam por adotar nosso sistema em vez de investir em uma plataforma própria.

Dentro do sistema, implementamos rigorosos controles de acesso para cada funcionário das empresas-clientes, garantindo permissões específicas para manter a segurança. A estrutura resultante oferece uma solução confiável e adaptável para o gerenciamento de estoque, atendendo tanto às necessidades das empresas contratantes quanto aos usuários internos da "MILL S.A".
