Sobre o projeto

Este foi meu primeiro projeto desenvolvido em React, uma página que exibe informações detalhadas sobre filmes utilizando a API do TheMovieDB.

O site permite a busca por filmes, mostrando resultados que incluem o cartaz, notas, data de lançamento, orçamento e outros detalhes relevantes.

Este projeto foi uma excelente oportunidade para aprender e praticar React, integrando APIs e manipulando dados em tempo real.
