Sobre o projeto

Este projeto foi desenvolvido durante meu período de estudo do framework Django, e fiquei bastante satisfeito com os resultados obtidos, especialmente considerando que foi minha primeira experiência nessa tecnologia. Aqui, utilizei uma combinação de HTML5, CSS3, JavaScript, Bootstrap e, é claro, o framework Django, juntamente com o banco de dados PostgreSQL.

A ideia do projeto era criar um site de receitas, que tem como propósito proporcionar a qualquer pessoa a oportunidade de se cadastrar e compartilhar suas próprias receitas. As receitas podem ser mantidas públicas para serem acessíveis a todos ou privadas, restritas apenas ao autor da receita. Cada usuário e suas respectivas receitas são armazenados de forma segura no banco de dados.
